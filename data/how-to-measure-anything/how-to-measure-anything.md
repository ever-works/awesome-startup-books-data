## Overview

How to Measure Anything shows that anything can be measured, especially the things thought to be immeasurable.

## Core Principles

1. If something matters, it's detectable
2. If it's detectable, it can be measured
3. Measurement reduces uncertainty
4. Even partial information has value

## Measurement Techniques

- Decomposition: Break complex things into measurable parts
- Random sampling
- Bayesian approaches
- Rule of Five
- Monte Carlo simulations

## Applications

- ROI of training programs
- Value of brand reputation
- Customer satisfaction metrics
- Innovation value
