# The Cold Start Problem: How to Start and Scale Network Effects

A startup playbook by Andrew Chen focused on how to launch and scale products that rely on network effects—covering early traction, growth loops, marketplace dynamics, and strategies for overcoming the “cold start” phase that kills many startups. Highly relevant for founders building platforms, marketplaces, and social or B2B networks.
